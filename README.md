# DragonAgeRPG-CC

Character Creator for Dragon Age Tabletop RPG

## Getting Started
Install the module with: `npm install DragonAgeRPG-CC`

```javascript
var DragonAgeRPG_CC = require('DragonAgeRPG-CC');
DragonAgeRPG_CC.awesome(); // "awesome"
```
